## 📌 Overview

This project represents my **first step into front-end web development**, where I built a fully responsive **e-commerce furniture website** from scratch using HTML, CSS, and JavaScript.

The goal of this project was to understand how real-world websites are structured and to practice building complete user interfaces that simulate an online shopping experience. The application includes multiple sections such as a hero landing area, featured and latest products, categories, promotional offers, and a blog section.

Throughout this project, I focused on learning and applying core front-end concepts such as semantic HTML structure, responsive layout design, and styling using CSS. I also worked on organizing content into reusable sections and improving visual consistency across the website.

Despite being an early project, it demonstrates my ability to:
- Structure a complete multi-section website
- Design clean and user-friendly interfaces
- Organize layouts using modern front-end practices
- Build a responsive design that works across different screen sizes

This project marks the beginning of my journey in web development and reflects my growing understanding of how to build scalable and visually appealing web applications.
